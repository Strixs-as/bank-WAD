⚡ Как это работает

Начните пользоваться за 4 простых шага

1

Регистрация

Создайте аккаунт с email и паролем. JWT-токен выдаётся автоматически.

2

Открыть счёт

Выберите тип (Checking/Savings/Investment) и валюту (RUB/USD/EUR).

3

Пополнить

Пополните баланс и начните совершать переводы между счетами.

4

Управлять

Выпускайте карты, оформляйте кредиты, открывайте вклады.

🔌 REST API

Base URL: http://localhost:8080

Все защищённые эндпоинты требуют заголовок: Authorization: Bearer <TOKEN>

POST/api/auth/register

Регистрация пользователя

POST/api/auth/login

Вход, получение JWT

GET/api/accounts

Список счетов пользователя

POST/api/accounts

Открыть новый счёт

POST/api/transactions/transfer

Перевод между счетами

POST/api/transactions/deposit

Пополнение счёта

GET/api/cards

Карты пользователя

POST/api/loans

Заявка на кредит

POST/api/deposits

Открыть вклад

Java EE REST демо
